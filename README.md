# Autonomie e Beni Comuni — Website

Sviluppo e manutenzione del sito istituzionale [autonomiebenicomuni.eu](https://autonomiebenicomuni.eu).

## Struttura
- `theme/` — Child theme WordPress
- `plugins/` — Plugin custom
- `docs/` — Documentazione

## Workflow
1. Sviluppo in locale su WordPress Studio
2. Test e revisione
3. Push su GitHub
4. Deploy sul sito live
